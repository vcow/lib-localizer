Detailed Description

A base class for the LocalizationManager.

TTextControllerA type of the controller component that adds to the UI Text object when Localize() method with the applyController flag is calling.
TTMPControllerA type of the controller component that adds to the UI TextMeshPro object /// when Localize() method with the applyController flag is calling.
