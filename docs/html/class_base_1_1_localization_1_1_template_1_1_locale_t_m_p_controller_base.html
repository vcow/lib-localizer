Detailed Description

A base class of the interactive localization controller for the UI TextMeshPro game object. Tracks changes of the current selected language and change the text in the TextMeshPro component respectively. Can receive the list of formatted string arguments. Use initial text of the TextMeshPro component as a localization key.
