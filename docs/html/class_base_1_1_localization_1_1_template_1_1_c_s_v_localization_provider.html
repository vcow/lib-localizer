Detailed Description

Localization provider for the ILocalizationManager that receive in input a CSV formatted strings where caption is a language abbreviation and left column is the string localization key. That provider works with the Unity resource assets as raw text sources and should receive that assets as a list of arguments when calling Init() method. If you wat to use some other sources, such as files or Internet resources override Init() method.
